# spring-boot-api-document
spring-boot-api-document 是一个基本Springboot 开发的一个接口文档工具，清晰的向前端开发员展示数据结构和数据标准，并具有后端数据校验功能。

# 技术要求
## 应用场景
1. **Springboot 项目，前后端分离项目**
2. **SpringCloud 项目，前后端分离项目**

## 环境需求
 1. **springboot** : 2.x；
 2. **jdk**:1.8；
